# UBST2508 Scheduler

Кроссплатформенное приложение для отслеживания учебного расписания и дедлайнов.

## Технологии
- **Язык:** C++20
- **Фреймворк:** Qt 6.8 (QML / Qt Quick)
- **Сборка:** CMake
- **Платформы:** Windows, Linux, Android, iOS

## Функционал
- Просмотр расписания на 2 недели.
- Карточки предметов с контактами преподавателей.
- Календарь дедлайнов с push-уведомлениями.
- Интеграция с Google Sheets (данные) и Telegram (файлы).

## Разработка
Проект находится на стадии проектирования архитектуры (MVP).
